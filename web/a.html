a menunjukkan sebuah anchor. Apa itu anchor? Anchor adalah teks yang menunjukkan awalan dan akhiran sebuah hypertext link.

HTML <a> element digunakan untuk mendefinisikan sebuah hyperlink.

Link dapat merujuk ke halaman yang sedang terbuka ataupun ke halaman lain (website lain).

Attribute target, download, rel, href dan type tidak boleh digunakan jika pada element a tersebut tidak disebutkan href attribute.
