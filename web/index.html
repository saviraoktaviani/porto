Materi

Element html

Tag pembuka dan penutup pada dokumen html

Element head

Tertera informasi dan keterangan yang berkaitan dengan dokumen yang dibuat

Element body

Isi dari sebuah dokumen HTML yang akan ditampilkan pada jendela browser.

Element heading

Element HTML yang digunakan untuk membuat judul artikel pada halaman HTML. Heading memiliki 6 tingkat (h1, h2, h3, h4, h5, h6)

Element p

Menunjukan sebuah paragraf

Element section

Digunakan untuk mengelompokkan konten/dokumen menjadi beberapa bagian berdasarkan tema atau pokok pikiran masing-masing.

Element division

Digunakan untuk menampung beberapa element menjadi satu kelompok.

Element link

Menunjukkan sebuah hubungan (link) antara dokumen (HTML) yang bersangkutan dengan sumber file dari luar (eksternal).

Element a herf

digunakan untuk mendefinisikan sebuah hyperlink. Link dapat merujuk ke halaman yang sedang terbuka ataupun ke halaman lain (website lain).

Element img

Untuk memsukan sebuah gambar pada dokumen HTML.
